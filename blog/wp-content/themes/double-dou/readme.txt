== Description == 
A simple content centered approached for Bloggers that want there visitors to only see what's important for the Blogger himself that he want to share to the world his/her blog posts and portfolio if there is any.

* Responsive Layout
* Social Links
* Hidden Side Navigation
* The GPL v2.0 or later license. 

== Installation ==
1. In your admin panel, go to Appearance -> Themes and click the Add New button.
2. Click Upload and Choose File, then select the theme's ZIP file. Click Install Now.
3. Click Activate to use your new theme right away.

== FAQ ==
1. How many child menus can I have?
- At most you can have 3 levels(Parent,sub-parent,sub-child), to consider the presentation
with regards how they are aligned.
- But the designs are quite easy to be edited just view the class via "view_source" then look
it up at the theme's style.css

2. I only want the background color not the image, How do I do that?
- You can do that by setting in the Theme Options > General(Tab) > Background Image or Pattern
set the text field to "none" w/o the ""

3. How to add new social links that aren't in the theme options below the Owner's bio?
- This kind of feature will be available in future versions of the theme. If you want just
contact me through my email.

4. How to add new templates based on the theme?
- There is a file in the theme's root directory: template-referrence.php
that is a blank template which you can use for your pages to have a similar look and feel
of the site.
